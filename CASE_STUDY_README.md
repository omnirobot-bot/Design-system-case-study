# Enterprise Design System Case Study

A high-end, interactive case study showcasing a multibrand, multiplatform design system architecture for enterprise cybersecurity products.

## Project Overview

This case study demonstrates:
- **Multibrand System**: Unified design system serving Norton, Avast, AVG, and Avira
- **Multiplatform Output**: Native implementations for Windows, macOS, iOS, and Android
- **Token Architecture**: Three-tier token system (Core → Semantic → Component)
- **Automation**: Supernova-powered design-to-code pipeline
- **AI Documentation**: Automated documentation generation
- **Scale**: Enterprise-grade system architecture

## Visual Design

### Style
- **Theme**: Premium dark theme with deep black (#070910) background
- **Surfaces**: Glassmorphism with semi-transparent surfaces and backdrop blur
- **Typography**: Inter for UI, JetBrains Mono for code/tokens
- **Accent Colors**:
  - Blue (#3B82F6) - System/Core layer
  - Purple (#A855F7) - Tooling/Automation
  - Green (#10B981) - Platform output
  - Yellow (#F59E0B) - Brand layer

### Components
- GlassCard - Glassmorphism container
- Badge - Multi-variant label system
- DiagramNode - Interactive diagram elements
- Navigation - Page navigation with keyboard support

## Pages

1. **Cover** - Introduction and project scope
2. **Context** - Problem, goals, and constraints
3. **Architecture** - System architecture diagram and flow
4. **Tokens** - Three-tier token hierarchy
5. **Multibrand** - Brand abstraction through tokens
6. **Multiplatform** - Platform-native implementations
7. **Components** - Component anatomy and state matrix
8. **Tooling** - Supernova pipeline and automation
9. **AI Docs** - AI-powered documentation workflow
10. **Outcomes** - Metrics, benefits, and learnings

## Features

- **Keyboard Navigation**: Use arrow keys (← →) to navigate
- **Smooth Transitions**: Subtle page transitions
- **Responsive Design**: Adapts to different screen sizes
